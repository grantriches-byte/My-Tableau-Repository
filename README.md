# Tableau Retail & Sales Dashboard Project

This project was completed as part of a **Data Technician Bootcamp** and focuses on analysing retail and sales data using **Tableau**. The goal of the project was to build interactive dashboards that help explore sales performance, customer trends, and regional insights through clear data visualisation.

The project demonstrates how Tableau can transform raw sales data into interactive visualisations that support data-driven decision making.

---

## Project Overview

Using a retail sales dataset, this project explores patterns in product performance, regional sales distribution, and customer behaviour. The dashboard allows users to interact with the data through filters and dynamic visualisations to uncover trends and insights.

The project focuses on combining multiple visualisations into a single dashboard to present a clear story about sales performance.

---

## Tableau Skills Demonstrated

This project demonstrates several key Tableau skills including:

- Creating **interactive dashboards**
- Building multiple chart types including:
  - Bar charts
  - Line charts
  - Pie charts
  - Map visualisations
- Using **filters and slicers** to allow users to explore the data
- Creating **calculated fields** to derive additional insights
- Combining multiple charts into a **single dashboard view**
- Designing dashboards that support **clear data storytelling**

---

## Data Visualisation & Storytelling

The dashboard visualises retail sales data to highlight:

- Sales performance by product category
- Regional sales patterns
- Trends in sales over time
- Comparisons across different segments of the data

These visualisations help transform raw data into insights that can be easily understood by business users.

---

## Dataset

The dataset contains retail and sales information including product categories, geographic regions, and sales metrics. This data was used to build visualisations that highlight trends and performance across different parts of the business.
